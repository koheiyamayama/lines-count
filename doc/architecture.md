# やること

- コマンドライン引数を受け取る。
  - オプション
    - `-p`
    - `-t`
  - パス
    - ファイル
    - ディレクトリ
- パスに指定されたものがファイルの場合
  - パスに指定されたファイルの行数を出力する
- パスに指定されたものがディレクトリの場合
  - パスに指定されたディレクトリに存在するファイルの合計の行数を出力する
  - `-p` を指定した場合はファイルごとの行数を出力する
  - `-t` を指定した場合はファイルの合計の行数を出力する
- パスに指定されたファイル or ディレクトリが存在しない場合
  - 標準エラー出力する
